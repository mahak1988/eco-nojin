# 📚 Econojin Digital Library

**پیشرفته‌ترین کتابخانه دیجیتال علمی جهان**

## ✨ ویژگی‌ها

- 📖 **کتابخانه دیجیتال**: کتاب، مقاله، پایان‌نامه، ویدیو
- 💬 **تالارهای گفتگو**: عمومی، خصوصی، تخصصی، قفل‌دار
- 🔬 **میز تحقیق**: همکاری تحقیقاتی real-time
- 🤖 **5 مشاور AI**: محیط زیست، بلاکچین، AI، کشاورزی، اقتصاد
- 🎓 **وبینارها**: کلاس‌ها و کارگاه‌های آنلاین
- 💎 **Eco Token**: سیستم پاداش بلاکچین (Polygon)
- 🌍 **20 زبان**: fa, en, ar, tr, zh, es, fr, de, ru, ja, ko, it, pt, nl, pl, sv, hi, ur, id, ms

## 🚀 شروع سریع

```bash
# Frontend
cd frontend && npm install && npm run dev

# Backend
cd backend && python -m venv venv
# Windows: venv\Scripts\activate
# Linux/Mac: source venv/bin/activate
pip install -r requirements.txt
python -m uvicorn api.main:app --reload

# Contracts
cd contracts && npm install
npx hardhat node
```

## 📊 تکنولوژی‌ها

- **Frontend**: Next.js 15.0.5, React 18, TypeScript, Tailwind CSS
- **Backend**: FastAPI, SQLAlchemy, PostgreSQL
- **Blockchain**: Solidity, Hardhat, Polygon
- **Database**: PostgreSQL with Supabase
- **Storage**: Cloudflare R2
- **AI**: Hugging Face
