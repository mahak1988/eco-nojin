# Operational Runbooks

## Runbook 1: Daily Health Check
- Check API health
- Verify all services running
- Check Grafana dashboards
- Review Prometheus alerts

## Runbook 2: Weekly Backup Verification
- Verify PostgreSQL backup exists
- Verify Redis backup exists
- Test backup restoration

## Runbook 3: Incident Response
- P1 (Critical): 15 minutes response
- P2 (High): 1 hour response
- P3 (Medium): 4 hours response
- P4 (Low): 24 hours response
