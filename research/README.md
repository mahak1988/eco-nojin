# Eco Nozhin Design Research - Phase 1

## Overview

Comprehensive research for the Eco Nozhin design system:
- 17 UN Sustainable Development Goals (SDGs)
- Natural elements from 7 world regions
- 12 sustainable agriculture techniques
- Complete design system foundation

## Structure

```
research/
  sdgs/sdgs-data.json         - 17 goals with colors
  sdgs/sdgs-colors.ts         - Official SDG colors
  nature/regional-symbols.json - 7 regions with symbols
  agriculture/sustainable-farming.json - 12 techniques
  design-system/colors.ts     - Color system
  design-system/typography.ts - Typography
  design-system/animations.ts - Animations
```
