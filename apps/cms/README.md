# Strapi CMS

This package contains the Strapi v5 headless CMS for managing marketing pages, blog posts, and tenant content.

## Running locally

```bash
pnpm dev:cms
```

## Notes

- Configure `PUBLIC_URL`, `DATABASE_URL`, and `ADMIN_JWT_SECRET` in your environment.
- Use Strapi content types for articles, pages, and tenant-specific marketing content.
